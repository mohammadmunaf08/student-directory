# Student Directory Portal

A beautiful and responsive React Student Directory built with Vite for Yenepoya Full Stack Web Development Assignment 2.

## Features
- Reusable `StudentCard` component with props
- Active & Alumni student sections using `.map()`
- Skills displayed with `.map()` inside each card
- Conditional Active/Inactive badges
- Directory statistics (total, active, alumni, most common skill)
- Fully responsive design
